Analytical Solute and Heat Transport Model (ASHTM) was developed for the U.S. Environmental Protection Agency in 2024 by APTIM and RTI. 
ASHTM has a graphical user interface to the transport executable (T3D). 
ASHTM executable, installation files, user manual, and supporting info are hosted on the Clu-In website: https://www.clu-in.org/conf/tio/ASHTM/
T3Di.1.4.3 is a new version of the executable (T3D) included in the 2024 installation.
